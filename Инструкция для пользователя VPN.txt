1. Инструкция для пользователя VPN
Цель: Подключиться к корпоративной сети через OpenVPN.

Шаги:

Установите клиент OpenVPN:

Windows: https://openvpn.net/client/

macOS: Tunnelblick

Linux: sudo apt install openvpn

Получите конфигурационный файл:

Администратор предоставит файл ваш_логин.ovpn.

Сохраните его в папку:

Windows: C:\Program Files\OpenVPN\config\

macOS: Просто откройте файл в Tunnelblick.

Подключитесь:

Запустите OpenVPN-клиент → выберите ваш профиль → нажмите "Connect".

Введите логин/пароль (если запросит).

Проверьте подключение:

Откройте Яндекс.Интернетометр https://yandex.ru/internet

Ваш IP должен совпадать с IP VPN-сервера.

Устранение неполадок:

Если подключение не работает:

Проверьте интернет-соединение.

Убедитесь, что файл .ovpn не поврежден.

Обратитесь к администратору: ****@****.***.
